# OBS用ストップウォッチ

OBS Studioのブラウザソースとして利用できる、カスタマイズ可能なストップウォッチです。

## 特徴

*   **透明な背景**: OBSのゲーム映像や配信画面に重ねて表示可能です。
    
*   **詳細なカスタマイズ**: タイトル、フォントサイズ、文字間隔、カラーを設定パネルから簡単に変更できます。
    
*   **スプリット機能**: 経過時間を記録し、画面上にリスト表示できます。
    
*   **視認性**: 配信中でも操作しやすいように、半透明で押しやすいボタンを採用しています。
    
    *   **OBS対話操作への最適化**: カラーピッカーが反応しない環境向けのプリセットボタンを搭載しています。
    

## 使い方

1.  **OBSでの追加方法**
    
    *   OBSの「ソース」欄の「＋」ボタンをクリックし、「ブラウザ」を選択します。
        
    *   URL欄にこのHTMLファイルを指定するか、ソースのプロパティからファイルを読み込んでください。
        
    *   幅と高さを適宜調整して画面上に配置します。
        
2.  **基本操作**
    
    *   **スタート/ストップ**: タイマーの計測を開始・一時停止します。
        
    *   **リセット**: タイマーとスプリット記録を初期化します。
        
    *   **スプリット**: 現在のタイムを記録します。
        
3.  **設定方法**

    *   画面下部の設定パネルから直接変更できます。

    *   **タイトルの変更**: テキスト入力後、**Enterキーを押す**か、**他の入力欄をクリック**（フォーカスを外す）すると反映されます。これはOBSの対話操作時における日本語入力（IME）を安定させるための仕様です。

    *   **色・サイズ・間隔の変更**: 数値を変更したり、カラーピッカーを操作した瞬間にリアルタイムで反映されます。

    *   **カラープリセット**: OS標準のカラーピッカーが反応しない場合は、横にある色付きのボタンをクリックすることで素早く色を切り替えられます。

## 注意事項

*   ブラウザソースのプロパティで「ソースがアクティブでないときにシャットダウンする」のチェックを外すと、配信中もタイマーが止まらずに維持される場合があります。
