# OBS用ストップウォッチ

OBS Studioで利用できる、カスタマイズ可能なストップウォッチツールです。「ブラウザソース版（HTML）」と「Luaスクリプト版」の2種類を同梱しています。

## 特徴

*   **ブラウザソース版 (HTML)**
    *   **透明背景**: 配信画面のどこにでも自由に配置可能です。
    *   **設定パネル**: 画面上のパネルでタイトル、サイズ、色、文字間隔を即座に調整できます。
    *   **スプリット機能**: 経過時間を記録し、リストとして画面上に残せます。
    *   **操作の安定化**: OBSの「対話」操作時の日本語入力（IME）トラブルを対策済みです。
    *   **カラープリセット**: カラーピッカーが動作しない環境でも、ボタン一つで主要な色に変更可能です。

*   **Luaスクリプト版**
    *   OBS標準の「テキスト (GDI+)」ソースの内容を直接書き換えます。
    *   ホットキー（ショートカットキー）による開始・停止・リセットが可能です。
    *   ソースのプロパティとスクリプト設定間の「タイトル双方向同期」機能を搭載しています。


## 使い方

### ブラウザソース版
1.  OBSのソース一覧から「ブラウザ」を追加し、`index.html` を指定します。
2.  ソースを右クリック ＞ **[対話]** を開いて操作します。
3.  **タイトルの変更**: テキスト入力後、**Enterキーを押す** か **他の入力欄をクリック** すると反映されます（入力中の予期せぬ更新を防ぐための仕様です）。
4.  **色の変更**: カラーピッカーが反応しない場合は、横にある色付きボタンをクリックしてください。

### Luaスクリプト版
1.  詳細は同梱の **`Lua_Script_Guide.md`** を参照してください。
2.  OBSの「ツール」 ＞ 「スクリプト」から `obs-stopwatch.lua` を読み込んで使用します。
3.  **ホットキー設定**: OBSの「設定」 ＞ 「ホットキー」から操作キーを割り当てられます。

## 注意事項

*   ブラウザソースのプロパティで「ソースがアクティブでないときにシャットダウンする」のチェックを外すと、配信中もタイマーが止まらずに維持される場合があります。
*   タイマー稼働中は、テキストソース側からのタイトル同期は一時的に停止します。
