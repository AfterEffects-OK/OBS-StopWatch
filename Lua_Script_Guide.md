# Luaスクリプト（obs-stopwatch.lua）設定ガイド

このスクリプトを使用すると、OBS標準の「テキスト (GDI+)」ソースを直接ストップウォッチとして動作させることができます。

## 1. インストール手順
1. OBS Studioを起動します。
2. 上部メニューの **[ツール]** ＞ **[スクリプト]** を選択します。
3. スクリプトウィンドウの左下にある **[＋]** ボタンをクリックします。
4. ファイル選択画面で `obs-stopwatch.lua` を選択して開きます。

## 2. ソースの作成と紐付け
スクリプトを動かす前に、表示用のテキストソースを準備する必要があります。
1. OBSの「ソース」パネルで **[＋]** ＞ **[テキスト (GDI+)]** を作成します（例：タイマー用、タイトル用）。
2. スクリプト設定画面の **[タイマー表示ソース]** と **[タイトル表示ソース]** のドロップダウンから、作成したソースを選択します。
   - **同じソースを指定した場合**: 「タイトル 00:00.0」のように1行にまとめて表示されます。
   - **別のソースを指定した場合**: それぞれ独立して表示されます。

## 3. ホットキー（ショートカット）の設定
配信中にスクリプト画面を開かずに操作するための設定です。
1. OBSの **[設定]** ＞ **[ホットキー]** を開きます。
2. 非常に多くの項目があるため、検索欄に「ストップウォッチ」と入力するか、リストをスクロールして以下の項目を探します。
   - **【スクリプト】ストップウォッチ：開始/停止**
   - **【スクリプト】ストップウォッチ：リセット**
3. 任意のキー（例：テンキーの「+」や「-」など）を割り当てて [OK] を押します。

## 4. タイトルの双方向同期について
このスクリプトには、入力の手間を省く同期機能があります。
- **スクリプトから変更**: スクリプト設定画面の「タイトル文字」を書き換えると、ソースに即座に反映されます。
- **ソースから変更**: OBSのソース一覧でテキストソースをダブルクリックし、プロパティで文字を書き換えて「OK」を押すと、スクリプト側の「タイトル文字」設定も自動的に更新されます。
  ※タイマー稼働中は同期が一時的に停止します。

## 5. うまく動かない場合
- **リストにソースが出ない**: スクリプト画面の右側にある **[スクリプトを再読み込み]** （丸い矢印ボタン）をクリックしてください。
- **文字が重なる**: 一度タイマーを停止させ、リセットボタンを押してから再度設定を確認してください。
